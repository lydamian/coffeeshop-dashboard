# How to Prepare a Cortado at RU Coffee Bar

A cortado is a balanced espresso drink of Spanish origin, consisting of equal parts espresso and steamed milk. The name "cortado" means "cut" in Spanish, referring to the milk cutting through the espresso.

## Cortado Calculator

export const CortadoCalculator = () => {
  const [espressoOz, setEspressoOz] = useState(2);
  const [steamedMilkOz, setSteamedMilkOz] = useState(2);
  const [totalVolume, setTotalVolume] = useState(4);

  useEffect(() => {
    setTotalVolume(espressoOz + steamedMilkOz);
  }, [espressoOz, steamedMilkOz]);

  const updateValues = (type, value) => {
    switch(type) {
      case 'espresso':
        setEspressoOz(value);
        setSteamedMilkOz(value);
        break;
      case 'steamedMilk':
        setSteamedMilkOz(value);
        setEspressoOz(value);
        break;
    }
  }

  return (
    <Card className="my-4">
      <CardHeader>
        <CardTitle>Cortado Recipe Calculator</CardTitle>
      </CardHeader>
      <CardContent>
        <div className="grid grid-cols-2 gap-4">
          <div>
            <label htmlFor="espresso-input">Espresso (oz):</label>
            <Input 
              id="espresso-input"
              type="number" 
              value={espressoOz} 
              onChange={(e) => updateValues('espresso', Number(e.target.value))}
              step="0.25"
              min="1"
              max="3"
            />
          </div>
          <div>
            <label htmlFor="steamedMilk-input">Steamed Milk (oz):</label>
            <Input 
              id="steamedMilk-input"
              type="number" 
              value={steamedMilkOz} 
              onChange={(e) => updateValues('steamedMilk', Number(e.target.value))}
              step="0.25"
              min="1"
              max="3"
            />
          </div>
        </div>
        <p className="mt-4">Total Volume: {totalVolume} oz</p>
      </CardContent>
    </Card>
  )
}

## Ingredients and Equipment

- Espresso (see <Link href="/recipes/espresso">Espresso Recipe</Link>)
- Whole milk
- Espresso machine with steam wand
- Milk pitcher
- 4-5 oz glass or ceramic cup

## Steps to Prepare a Cortado

1. **Prepare the espresso**
   - Follow the <Link href="/recipes/espresso">Espresso Recipe</Link> to pull a 2 oz double shot

2. **Steam the milk**
   - Refer to the <Link href="/recipes/steamed-milk">Steamed Milk Guide</Link> for technique
   - Steam 2 oz of milk to 140-150°F (60-65°C)
   - Aim for minimal foam, just enough to create a velvety texture

3. **Combine and serve**
   - Pour the espresso into a 4-5 oz cup
   - Gently pour the steamed milk over the espresso
   - Aim for a 1:1 ratio of espresso to steamed milk

## Tips for the Perfect Cortado

1. **Espresso quality**: Use freshly roasted, high-quality beans and dial in your grind and extraction
2. **Milk texture**: Focus on creating silky, lightly textured milk without significant foam
3. **Temperature**: Serve immediately for the best flavor and temperature
4. **Balance**: Maintain the 1:1 ratio of espresso to steamed milk for a true cortado
5. **Glassware**: Traditionally served in a small glass to showcase the layers, but ceramic cups are also common in specialty shops

Remember, a cortado is about balance and simplicity, allowing both the espresso and the milk to shine without overpowering each other.
